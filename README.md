# Admisoft
Aplicatia are ca scop parcurgerea unor chestionare intocmite dupa modelul concursului de admitere in Academia de politie, mai exact grila, ce se pot introduce foarte usor in baza de date. Sustinerea se realizeaza pe calculator, aplicatia fiind realizata initial pentru sistemul Windows, dar in cel mai scurt timp va cunoaste si versiunile Android si iOS. Trimiterea raspunsurilor se efectueaza intr-o maniera foarte simpla si nu exista loc de discutii. Testul se corecteaza automat, iar elevul isi poate vedea nota imediat dupa sustinerea lui.
